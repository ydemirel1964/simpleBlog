INSTALLATION

$ git clone https://github.com/ydemirel1964/simpleBlog.git <br>
$ cd simpleBlog        <br>
$ cp .env.example .env <br>
$ composer install <br>
$ php artisan key:generate<br>
$ npm install<br>
$ npm run build<br>
<br>
Before starting
<br><br>
You need to create simpleBlog database and after run the migrations:<br>
<br>
$ php artisan migrate
